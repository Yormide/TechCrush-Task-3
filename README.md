Project Overview
This project analyzes a year's worth of sales data from a fictitious pizza place. The goal was to transform raw CSV data into actionable business insights using Python. By merging multiple datasets, I explored sales trends, identified peak operating hours, and evaluated menu performance to provide data-driven recommendations.

The Dataset
The analysis is based on four relational CSV files:

Orders: Date and time of every order placed.

Order Details: Specific pizzas and quantities included in each order.

Pizzas: Size and price information for each pizza variant.

Pizza Types: Descriptive data including names, categories, and ingredients.

Technologies Used
Python 3.x

Pandas: For data cleaning, merging, and manipulation.

Matplotlib: For data visualization.

Jupyter Notebook: For interactive analysis and documentation.

4. Monthly Trends
Sales remained relatively stable throughout the year, with a slight peak in July and a minor dip in the autumn months (September/October).

Strategic Recommendations
Staffing: Increase kitchen staff during the 12 PM and 6 PM rushes, especially on Fridays.

Menu Optimization: Consider removing or rebranding the Brie Carre Pizza to reduce inventory waste.

Promotions: Introduce "Family Bundle" deals on Sundays to boost revenue during the slowest day of the week.
